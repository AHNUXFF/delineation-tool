Interactive delineation is done with the QGIS plugin BoundaryDelineation:
https://plugins.qgis.org/plugins/BoundaryDelineation/

The source code is available here:
https://github.com/SCrommelinck/BoundaryDelineation

Instructions on how to apply interactive delineation can be found in the wiki:
https://github.com/SCrommelinck/Delineation-Tool/wiki/3)-Interactive-Delineation

The two folders available here offer two appraoches for one substep (simplify segmentation), 
which can be run in QGIS or ArcGIS. Instructions on when and how to apply one of the approaches can be found in the wiki.

A video demonstrating the installation and use of the QGIS plugin BoundaryDelineation ca be found on YouTube:
https://www.youtube.com/watch?v=GrDv8fW53Fs
